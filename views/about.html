What are Rave Squares?


A Rave square is a small square with some CSS and Javascript info. That's all! Oh, and details? Well...



"Ok, cool," you're saying, "but what the heck is going on behind the scenes!?"

When this page loads, it calls a javascript function that attaches an element with height and width, but no color. A timer alternates betwen a random color for that square. When you assemble a bunch of these squares into a table, it looks like you've created your very own rave. Woo hoo!
